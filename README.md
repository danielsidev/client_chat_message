# ReactJS Client Message

CLiente web em ReactJS para o chat message.

### Instale as dependências 
``` bash
sudo npm install
```
### Funcionamento do client:
Rode o comando abaixo, após ter inicializado a aPI do chat message.
``` bash
sudo npm run start
```
Informe um nome para se identificar no chat e após entrar, pode começar a participar do chat.
---

## Host: http://localhost:3000 

Para rodar em produção, faça o build do cliente e o push do build public.

``` bash
sudo npm run build
```
